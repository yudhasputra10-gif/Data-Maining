# Data Mining Python Project

Proyek ini merupakan implementasi **Data Mining menggunakan Python** untuk melakukan proses analisis data, preprocessing, visualisasi, dan prediksi menggunakan algoritma machine learning.

## 📌 Fitur
- Data preprocessing
- Data cleaning
- Visualisasi data
- Training model machine learning
- Evaluasi model
- Prediksi data

## 🛠️ Teknologi yang Digunakan
- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
